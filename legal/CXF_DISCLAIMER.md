# CxF Export Disclaimer

The CxF (Color Exchange Format) export feature provided in the Premium version of LabSwatch is intended as a **basic custom palette exchange format**.

## Important Limitations:
1. **No Licensed Data:** The exported CxF files do not include, use, or reference any licensed proprietary color data, libraries, or names from Pantone, X-Rite, or any other trademarked color systems.
2. **Calculated Values:** The values included in the CxF export (sRGB, XYZ, LAB) are computationally derived by the app assuming an sRGB input and a D65 illuminant.
3. **Not Certified:** This export is not certified for production proofing, and it lacks true spectrophotometric spectral data.
4. **Verification Required:** For color-critical physical production, always verify your target colors with calibrated hardware and industry-standard reference materials.
